# Remaining Tasks

## Immediate

- Create or connect the public GitHub repository `yangpane/DogPet`.
- Add `TAURI_SIGNING_PRIVATE_KEY` and optional `TAURI_SIGNING_PRIVATE_KEY_PASSWORD` secrets.
- Push a `v1.1.0` tag and inspect the draft GitHub Release.
- Test the release DMG on a clean macOS install flow.
- Test the Windows NSIS installer from GitHub Actions.

## MVP Completion

- Decide whether to keep the original Live2D model assets in the public v1 package or remove them later to shrink installer size.
- Add a first-run check that reports missing bundled assets in a user-friendly way.
- Verify updater behavior with two test releases.

## Later

- Video support.
